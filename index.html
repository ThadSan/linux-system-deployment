Linux System Deployment & Web Server Setup

SAÉ 1.03 — Installation d’un poste pour le développement

Ce site présente la mise en œuvre complète d’un environnement Linux virtualisé, depuis l’installation du système jusqu’au déploiement d’un serveur web Apache fonctionnel.

Contexte du projet

Dans le cadre de la SAÉ 1.03, l’objectif était de comprendre et de mettre en pratique les bases de l’administration système sous Linux. Le projet repose sur la création d’une machine virtuelle Debian, la gestion des utilisateurs et des droits, ainsi que l’installation et la configuration d’un service réseau.

L’ensemble du travail a été réalisé manuellement, sans automatisation ni outil de génération, afin de garantir une compréhension réelle des mécanismes mis en jeu.

Travail réalisé

Virtualisation

Création et configuration d’une machine virtuelle sous Debian GNU/Linux à l’aide de VMware Workstation.

Administration système

Gestion des comptes utilisateurs, des groupes, des privilèges et des bonnes pratiques de sécurité.

Serveur web

Installation et déploiement d’un serveur Apache capable de servir des pages HTML via HTTP.
